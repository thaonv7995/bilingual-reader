Truy vấn Dòng Top-N

PostgreSQL

PostgreSQL hỗ trợ phần mở rộng fetch first từ phiên bản 8.4. Câu lệnh limit trước đây vẫn hoạt động như trong ví dụ MySQL.

SQL Server

SQL Server cung cấp câu lệnh top để giới hạn số lượng dòng được lấy.

Bắt đầu từ phiên bản 2012, SQL Server cũng hỗ trợ phần mở rộng fetch first.

Tất cả các câu truy vấn SQL trên đều đặc biệt vì các cơ sở dữ liệu nhận diện chúng như các truy vấn top-N.
