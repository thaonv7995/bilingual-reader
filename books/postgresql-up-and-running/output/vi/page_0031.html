PostgreSQL 9.1: Tính Năng Mới

Với phiên bản 9.1, PostgreSQL đã triển khai các tính năng doanh nghiệp để cạnh tranh trực tiếp với các tên tuổi lớn như SQL Server và Oracle:

  • Nhiều tính năng sao chép tích hợp hơn, bao gồm sao chép đồng bộ.
