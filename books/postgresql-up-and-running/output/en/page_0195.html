Using Filter Instead of CASE

PostgreSQL 9.4 offers the new FILTER construct, which we introduced in “FILTER Clause for Aggregates” on page 131. FILTER can often replace CASE in aggregate ex­pressions. Not only is this syntax pleasanter to look at, but in many cases performs better. We repeat Example 9-15 with the equivalent filter version in Example 9-16.
