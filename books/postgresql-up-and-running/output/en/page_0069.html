Although \watch is primarily designed for monitoring query output, you can also use it to run statements at fixed intervals. In Example 3-4, we log activity every five seconds.

① Create table and perform first insert.

② Insert every five seconds.

If you want to kill a watch, use CTRL-X CTRL-C. Needless to say, watches are meant for interactive psql only.

You can find more examples of using watch at Michael Paquier: Watch in psql.
