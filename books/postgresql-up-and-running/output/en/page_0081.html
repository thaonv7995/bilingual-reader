For example, to set privileges for items in the census schema, select the schema and then choose the Grant Wizard. The Grant Wizard screen is shown in Figure 4-7. You can then select all or some of the items and switch to the Privileges tab to set the roles and privileges you want to grant.

More often than setting privileges on existing objects, you may want to set default priv‐ ileges for new objects in a schema or database. To do so, right-click the schema or database, select Properties, and then go to the Default Privileges tab, as shown in Figure 4-8. Default privileges are available only for PostgreSQL 9.0 and later.
