String Functions

Common string manipulations are padding (lpad, rpad), trimming whitespace (rtrim, ltrim, trim, btrim), extracting substrings (substring), and concatenating (||). Example 5-2 demonstrates padding, and Example 5-3 demonstrates trimming.

  1. ❶ lpad truncates instead of padding if string is too long.

By default, trim functions remove spaces, but you can pass in an optional argument indicating other characters to trim.

Version 9.0 introduced a helpful string aggregate function called string_agg, which we demonstrate in Example 3-11 and Example 5-21. string_agg is equivalent in concept to the group_concat function in MySQL.
