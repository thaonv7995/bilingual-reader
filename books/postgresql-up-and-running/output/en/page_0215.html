Server Backup: pg_dumpall

Use pg_dump_all to back up all databases on your server onto a single plain-text or plain-text SQL file. The backup routine will automatically include server-level objects such as roles and tablespaces. Example B-2 shows the pg_dumpall help output. See “Systemwide Backup Using pg_dumpall” on page 40 for the full discussion.
