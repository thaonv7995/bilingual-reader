PostgreSQL 9.3: New Features

The notable features that first appeared in version 9.3 (released in 2013) are:

  • The ANSI SQL standard LATERAL clause was added. A LATERAL construct allows FROM clauses with joins to reference variables on the other side of the join. Without this, cross-referencing can take place only in the join conditions. LATERAL is indis‐ pensable when you work with functions that return sets, such as unnest, generate_series, regular expression table returns, and numerous others. See “Lateral Joins” on page 139.
  • Parallel pg_dump is available. Version 8.4 brought us parallel restore, and now we have parallel backup to expedite backing up of huge databases.
