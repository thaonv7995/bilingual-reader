Therefore, it refuses to compile the code.

So, to summarize, the type arguments of the parameters of the overriding method must also match exactly with the type arguments of the parameters of the overridden method.

Covariance with parameterized return types

As discussed in the chapter on Inheritance, an overriding method is allowed to return a covariant return type. Meaning, if a base class’s method returns List, the overriding method in the subclass may return ArrayList because ArrayList is a subtype of List.
