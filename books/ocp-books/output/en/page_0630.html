So, LocalDateTime.now() just picks up the current date and time values from the local machine and ignores the time zone. LocalDate.now() and LocalTime.now() work similarly.

OffsetDateTime.now() and ZonedDateTime.now() do the same thing except that they also pick up the system clock’s offset and time zone information respectively.

The Instant class, on the other hand, represents the number of milliseconds elapsed since epoch (remember?) and so, Instant.now() returns the an Instant object that contains the number of milliseconds elapsed since epoch as of the moment that Instant object is created.
