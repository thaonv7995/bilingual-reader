17.1.7 Scope of variables in a lambda expression☝

There is just one more thing that you need to know about lambda expressions. The variables that you define in the variable section of a lambda expression exist in the same scope as which the lambda expression itself exists. It is possible to access a variable that is in scope of the lambda expression from within the lambda expression's body but only if that variable is declared as final or is “effectively final”. Effectively final means that even though the variable is not explicitly declared as final, its value is not changed though out the scope in which it exists, and so, the compiler assumes it as final.
