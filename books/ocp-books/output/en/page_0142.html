  1. a.mash(la) : The compiler notices that the declared type of a is Apple . It will therefore, consider only the methods available in class Apple to bind this invocation. It also notices that the declared type of the argument la is List . Apple class has two methods that can accept a List - mash(Collection ) and mash(List ) . Since the declared type of argument la matches exactly with mash(List ) , it binds the call to Apple’s mash(List ) method.

    At run time, the JVM notices that a points to an object of class Apple and so, it invokes Apple’s mash(List ) . Thus, it prints In Apple’s mash(List ) .
