24.3.5 Formatting Messages using MessageFormat☝

Showing messages that are grammatically correct in different locales is tricky. For example, a sentence such as "Your balance is 100 points." would be grammatically correct in many languages such as English. However, it would be grammatically incorrect in many other languages such as Hindi, in which the correct form would be, "Your balance 100 points is".

Thus, if the program creates this message using string concatenation such as String msg = rb.getString("yourbalanceis")+" "+amount+" "+rb.getString("points");, the resulting string may look odd in some locales.
