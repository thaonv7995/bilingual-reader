The Console class has the following two methods to print text on the console:

  1. Console format(String format, Object... args) - Writes a formatted string to this console’s output stream using the specified format string and arguments. For example, the following code prints, Hi Bob, how are you? :

Complete details of the format specifiers are not required for the exam but knowing about a few commonly used specifiers such as %s , %d , %f , and %t can be helpful while writing professional code.

  1. Console printf(String format, Object... args) - This is a convenience method that works exactly the same way as the format method.

Observe that the above two methods return a reference to a Console. It is a reference to the same Console object on which the methods are invoked. The purpose of the return value is to enable chaining multiple method calls in the same statement. For example, c.format("Hi! ").format("Good Morning");
