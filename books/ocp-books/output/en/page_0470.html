Using Thread.Builder to create threads☝

The way of creating threads discussed above is the pre-Java 21 way of creating threads. It remains valid even now in Java 21 and works the same as before. But it is meant to create platform threads only. Now that Java 21 has introduced virtual threads, a new Thread.Builder interface has been defined in the Java standard library for creating platform as well as virtual threads. The Thread class too has been enhanced to include two new static methods named ofPlatform and ofVirtual that return Thread.Builder objects to create platform and virtual threads respectively.
