Which of the following options correctly instantiate the interface X using a lambda expression?

Select 1 correct option.

A. X x = (List l) -> l.size();

B. X x = (ArrayList l) -> l.size();

C. X x = l -> l.toString();

D. X x = l -> { };

E. X x = l -> l;

Correct answer is A, C, E.

This question tests your knowledge of the rules of implementing an interface. A class that implements X must have a method whose parameter type matches exactly with the parameter type of the method in the interface (i.e. List) and whose return type is covariant with return type specified in the interface method (i.e. Object). Thus, A is correct but B is not. A, C, and E are correct because int (which will be boxed into an Integer), String and List are covariant with Object and can be returned from this method. D is incorrect because the method m is expected to return a value but this lambda expression returns void.
