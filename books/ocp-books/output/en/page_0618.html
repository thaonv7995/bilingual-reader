21.3.6 Creating primitive streams using ThreadLocalRandom☝

You have already seen the usage of Math.random() method generate random numbers. This method is properly synchronized and can be used safely from multiple threads. However, due to strong synchronization, it performs poorly when invoked from multiple threads simultaneously.
