If the value of the expression doesn’t match with any of the case labels, the control looks for a block labelled default and enters that block. If there is no default block either, the control does not enter the switch block at all. Since this “switching” is done based on the expression specified in the switch statement (which is just args.length in this example), this expression is aptly called the “selector expression”.
