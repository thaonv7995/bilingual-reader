Using the MessageFormat class is quite simple. The parameters are specified using curly braces and besides the parameter index (indexing starts with zero), you may specify how you want to format it as well. For example, a raw message could be, "On {0, date, short}, your balance was {1, number, currency}" and it can be formatted using an array of two objects as arguments like this: MessageFormat.format(msg, new Object[]{new Date(), 100});

This will return "On 10/05/24, your balance was $100.00.".

A serious issue with MessageFormat is that it was created before the introduction of the java.time package and hasn't been been updated to work with the new LocalDate/Time classes. So, if you want to use dates as arguments, you have to use java.util.Date objects.
