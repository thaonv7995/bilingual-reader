Generally, reading data byte by byte is not too useful. For example, if you want to manipulate image data, you would need to load the complete image file in program memory and then process it. Similarly, writing to a file one byte at a time is quite inefficient. InputStream and OutputStream have a few methods that help you perform I/O operations in bulk. To write the bytes to the output stream, we are using OutputStream 's write(byte[] b, int offset, int len) method, which writes len  number of bytes from the given byte array starting from the index given by the offset parameter (which is 0 in our example). Note that the number of bytes read may be less than 1024 in the last iteration of the while loop and in that case, the byte array will still have old values read in the previous iteration at the index position indicated by bytesRead and beyond. Since we are only writing bytesRead number of values starting from index 0 , those old values will not be written to the output stream.
