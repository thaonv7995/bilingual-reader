The truncatedTo method

The truncatedTo method returns a copy of the original date-time with fields smaller than the specified unit set to zero. For example, truncating with the DAYS unit will set the hours, minutes, seconds, and nano fields to zero:

Although not important for the exam, be aware that not all date-time classes support truncation with all available units. For example, ZonedDateTime doesn’t support truncation with units greater than DAYS.
