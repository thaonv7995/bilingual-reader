The selector expression☝

The expression written within switch(<expression>) is called the selector expression. Its value determines which case block will be executed. Prior to Java 21, the switch’s selector expression was restricted to only a few types (namely, byte, short, char, int and their wrapper types, String , and enums) but with Java 21 the only restriction that remains is that a selector expression must not return long , float , double , or boolean . To be more explicit, as of Java 21, the following types are allowed:

1. a limited set of integral types ( byte, char, short, int ), and their wrapper classes.

Observe that even though long is an integral type, it cannot be the type of a switch variable. boolean, float, and double are not integral types anyway and therefore, cannot be the type of a switch variable either.

2. enum type

3. java.lang.String - Since Java 7, you may use a String expression as a selector expression.
