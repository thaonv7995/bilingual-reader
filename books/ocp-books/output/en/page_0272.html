  1. <T> T[] toArray(T[] a) : Returns an array of the type specified in the argument containing all the elements of this list. The objects of the list must be of the type passed in the argument. This method is helpful when you want to convert a list into an array of a specific type. For example, String[] strArray = listOfString.toArray(new String[0]); will return a String[] containing all the elements of the list.
