13.6.3 Method selection with overloading and overriding☝

Before going through this section, make sure you understand the rules of method selection discussed in Chapter 10 (Section 10.2.3). Let me recap the cardinal principle that you should always keep in mind here:

While determining which method should be invoked, the compiler considers only the declared type of the variable on which the method is invoked and the declared type of the variables passed as arguments. At runtime, the JVM picks up the method signature that was selected by the compiler, finds out the actual object pointed to by the reference variable, and invokes the method on that object.
