Besides the above, there are readAttributes, getAttribute and getFileAttributeView methods to read file attributes. For example:

Different file systems support different file attributes but all of them support a few basic attributes that are common to all file systems. These are represented by the BasicFileAttributes interface. Others such as DosFileAttributes and PosixFileAttributes extend BasicFileAttributes. Although they are not explicitly mentioned in the exam, they are quite useful professionally.

Other utility methods in Files class☝

There are several methods to create, copy, move, and delete files as well as directories in the Files class. I am not listing them here because the exam doesn't quiz you on those although they may be used incidentally in code snippets. I suggest you to browse through the JavaDoc of the Files class to be aware of such methods.
