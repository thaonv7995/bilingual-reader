19.1.6 Advanced I/O stream operations☝

Rewinding a stream☝

Generally, data from an input stream can only be read once. You cannot "rewind" a stream and reread the same data again. However, some input streams that use internal data buffers may be able to provide this functionality. This functionality comes in handy while implementing parsers that need to rewind to a previous location if they encounter a specific control character.
