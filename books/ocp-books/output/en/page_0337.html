17.2 Method References☝

First off, the word reference in “method reference” does not mean the same as in object reference. A method reference does not “refer to” or “point to” a method like a reference variable does to an object because a method is not an object in Java. The word reference here just means indication.  As per JLS, a method reference expression is used to refer to the invocation of a method without actually performing the invocation.
