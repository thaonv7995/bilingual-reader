sorting order determined by the comparator. The java.util.Comparator interface has been around since Java 1.2 but it has been made a functional interface in Java 8. Its functional method is int compare(T o1, T o2) , which compares its two arguments and returns a negative integer, zero, or a positive integer if the first argument is less than, equal to, or greater than the second.
