Locale overrides the toString method to return a string representation of the Locale. As per JavaDoc, it returns language + "_" + country + "_" + (variant + "_#" | "#") + script + "_" + extensions. Language is always lower case, country is always upper case, script is always title case, and extensions are always lower case. In the exam, you will see Locales that use only the language and the country and their string representations will be like en_US or fr_CA.
