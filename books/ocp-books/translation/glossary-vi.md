# Glossary — Vietnamese (vi)

## Nguyên tắc chung

- **Giữ nguyên (English)**:
  - Tất cả mã nguồn trong `<pre class="code-block">` và các thẻ `<code>`.
  - Các từ khóa Java: `try`, `catch`, `finally`, `var`, `class`, `interface`, `enum`, `record`, `throw`, `throws`, `new`, `null`, `true`, `false`, `public`, `private`, `protected`, `void`, `return`, `static`, `final`, etc.
  - Tên các Class, Interface, Method, Variable, Package, Module (ví dụ: `String`, `List`, `ArrayList`, `AutoCloseable`, `close()`, `main()`, `java.io`, etc.).
  - Các ký hiệu trích dẫn, sơ đồ PNG/SVG (giữ nguyên đường dẫn `src`).
  - Các cụm từ viết tắt phổ biến: OCP, JVM, JDK, JRE, API, Garbage Collection (GC), JVM, API, CLI.

- **Dịch sang tiếng Việt**:
  - Phần văn bản giải thích (prose), tiêu đề chương, tiêu đề mục (`h3.section-title`, `.section-title`, etc.).
  - Nhãn điều hướng trang (Nav): `← Prev` → `← Trước`, `Next →` → `Tiếp →`, `Page N / M` → `Trang N / M`.
  - Các phần mô tả hình ảnh (`figcaption`), mô tả bảng.
