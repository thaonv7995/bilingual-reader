# Glossary — Vietnamese (vi)

## Giữ nguyên (English)

- Tên tác giả: Robert C. Martin
- URL: www.objectmentor.com
- Copyright line (footer)
- Mã nguồn C++ trong `<pre class="code-block">` — toàn bộ
- Tên class/method: Circle, Ellipse, Modem, User, Base, Derived, …
- Trích dẫn: [Martin99], [GOF96], [Shaw96], [LSP97], …
- Thuật ngữ viết tắt: OCP, LSP, DIP, ISP, DBC, ADP, SDP, SAP, REP, CCP, CRP, …
- Figure / Listing numbers: Figure 2-15, Listing 2-5 (giữ "Figure"/"Listing" hoặc dùng "Hình"/"Đoạn mã" — thống nhất **Hình** / **Đoạn mã**)
- Hình ảnh PNG/SVG: không sửa file, giữ `src` path

## Dịch sang tiếng Việt

- Prose, headings, run-in headings, figcaption mô tả
- Nav: Page → Trang, Prev → Trước, Next → Tiếp
- section-title block headings
- Footnotes giải thích (giữ mã trích dẫn)
