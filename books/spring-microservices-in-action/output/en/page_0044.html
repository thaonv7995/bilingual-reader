While we’ve chosen to use Spring Boot and Spring Cloud to implement the patterns we’re going to use in this book, nothing will keep you from taking the concepts presented here and using them with other technology platforms.   • Routing patterns
  • Client resiliency patterns

  • Build and deployment patterns
