JavaScript Web Tokens và OAuth2 221

(đã đề cập trước đó trong hình 7.10.) Đây không phải là trường JWT token tiêu chuẩn. Đó là trường tôi đã thêm bằng cách inject một trường mới vào JWT token khi nó đang được tạo.

Mở rộng JWT token rất dễ thực hiện bằng cách thêm class Spring OAuth2 token enhancer vào dịch vụ xác thực của bạn. Mã nguồn của class này nằm trong class authentication-service/src/main/java/com/thoughtmechanix/authentication/security/JWTTokenEnhancer.java. Listing sau đây trình bày code này.
