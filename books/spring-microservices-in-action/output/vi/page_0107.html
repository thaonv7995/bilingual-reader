82 CHƯƠNG 3 Kiểm soát cấu hình với Spring Cloud configuration server

Vì bạn đã bổ sung khả năng introspection cho tất cả các dịch vụ thông qua Spring Boot Actuator, bạn có thể xác nhận môi trường đang chạy bằng cách gọi http://localhost:8080/env. Endpoint /env sẽ cung cấp danh sách đầy đủ thông tin cấu hình về dịch vụ, bao gồm các thuộc tính và endpoint mà dịch vụ đã khởi động cùng, như minh họa trong hình 3.7.

Điểm quan trọng cần lưu ý từ hình 3.7 là profile đang hoạt động của licensing service là dev. Bằng cách kiểm tra JSON trả về, bạn cũng có thể thấy cơ sở dữ liệu Postgres được trả về là URI phát triển jdbc:postgresql://database:5432/eagle_eye_dev.

Mục environment trong tệp chứa giá trị của hai biến PROFILE, là Spring Boot profile mà licensing service sẽ chạy dưới đó. CONFIGSERVER_URI được truyền tới licensing service và xác định instance Spring Cloud configuration server mà dịch vụ sẽ đọc dữ liệu cấu hình từ đó.

Trong các script khởi động được container chạy, bạn sau đó truyền các biến môi trường này dưới dạng tham số –D cho các JVM khởi động ứng dụng.
