3.4.1 Tải xuống và cài đặt các jar Oracle JCE cần thiết cho mã hóa

Để bắt đầu, bạn cần tải xuống và cài đặt Unlimited Strength Java Cryptography Extension (JCE) của Oracle. Gói này không có sẵn qua Maven và phải được tải từ Oracle Corporation.1 Sau khi đã tải các tệp zip chứa các jar JCE, bạn phải thực hiện như sau:

  1. Xác định thư mục $JAVA_HOME/jre/lib/security của bạn.
  2. Sao lưu các tệp local_policy.jar và US_export_policy.jar trong thư mục $JAVA_HOME/jre/lib/security sang một vị trí khác.
  3. Giải nén tệp zip JCE bạn đã tải từ Oracle.
  4. Sao chép local_policy.jar và US_export_policy.jar vào thư mục $JAVA_HOME/jre/lib/security của bạn.
  5. Cấu hình Spring Cloud Config để dùng mã hóa.

Tự động hóa quy trình cài đặt các tệp JCE của Oracle

Tôi đã hướng dẫn các bước thủ công để cài JCE trên laptop của bạn. Vì chúng ta dùng Docker để build tất cả các dịch vụ thành container Docker, tôi đã viết script tải xuống và cài đặt các tệp JAR này trong container Docker Spring Cloud Config. Đoạn shell script OS X sau cho thấy cách tôi tự động hóa việc này bằng công cụ dòng lệnh curl (https://curl.haxx.se/):

Tôi sẽ không đi qua mọi chi tiết, nhưng về cơ bản tôi dùng CURL để tải các tệp zip JCE (lưu ý tham số header Cookie được truyền qua thuộc tính -H trên lệnh curl) rồi giải nén các tệp và sao chép chúng vào thư mục /usr/lib/jvm/java-1.8-openjdk/jre/lib/security trong container Docker của tôi.

Nếu bạn xem tệp src/main/docker/Dockerfile trong mã nguồn của chương này, bạn có thể thấy ví dụ về script này đang hoạt động.
