76 CHƯƠNG 3 Kiểm soát cấu hình với Spring Cloud configuration server

ứng dụng sẽ có các thuộc tính được quản lý bởi máy chủ cấu hình. Trong ví dụ trước, bạn chỉ cấu hình licensing service.

LƯU Ý Hãy lưu ý rằng nếu bạn dùng phiên bản Spring Cloud Config dựa trên hệ thống tệp cục bộ, bạn sẽ cần sửa thuộc tính spring.cloud.config.server.native.searchLocations để phản ánh đường dẫn tệp cục bộ của bạn khi chạy mã cục bộ.

Bạn đã hoàn thành đủ công việc để khởi động máy chủ cấu hình. Hãy khởi động máy chủ cấu hình bằng lệnh mvn spring-boot:run. Máy chủ sẽ hiển thị màn hình splash của Spring Boot trên dòng lệnh. Nếu bạn truy cập http://localhost:8888/licensingservice/default trên trình duyệt, bạn sẽ thấy payload JSON được trả về với tất cả các thuộc tính có trong tệp licensingservice.yml. Hình 3.4 cho thấy kết quả khi gọi endpoint này.

Nếu bạn muốn xem thông tin cấu hình cho môi trường licensing service dựa trên dev, hãy gọi endpoint GET http://localhost:8888/licensingservice/dev. Hình 3.5 cho thấy kết quả khi gọi endpoint này.

Nếu bạn nhìn kỹ, bạn sẽ thấy rằng khi gọi endpoint dev, bạn nhận về cả các thuộc tính cấu hình mặc định cho licensing service và cấu hình licensing service dev.
