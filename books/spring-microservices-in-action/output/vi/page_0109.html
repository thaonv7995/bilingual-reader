Lớp License là lớp model sẽ chứa dữ liệu lấy từ cơ sở dữ liệu licensing của bạn. Listing sau đây trình bày mã cho lớp License.

@Entity báo cho Spring biết đây là một class JPA.

@Table ánh xạ tới bảng cơ sở dữ liệu.

@Id đánh dấu trường này là khóa chính.

@Column ánh xạ trường tới một bảng cơ sở dữ liệu cụ thể.

Lớp này dùng một số Java Persistence Annotations (JPA) giúp framework Spring Data ánh xạ dữ liệu từ bảng licenses trong cơ sở dữ liệu Postgres sang đối tượng Java. Annotation @Entity cho Spring biết POJO Java này sẽ ánh xạ các đối tượng chứa dữ liệu. Annotation @Table cho Spring/JPA biết bảng cơ sở dữ liệu nào cần được ánh xạ. Annotation @Id xác định khóa chính của cơ sở dữ liệu. Cuối cùng, mỗi cột từ cơ sở dữ liệu sẽ được ánh xạ tới các thuộc tính riêng lẻ đều được đánh dấu bằng thuộc tính @Column.

Framework Spring Data và JPA cung cấp các phương thức CRUD cơ bản để truy cập cơ sở dữ liệu. Nếu bạn muốn xây dựng các phương thức vượt ngoài phạm vi đó, bạn có thể dùng interface Spring Data Repository và các quy ước đặt tên cơ bản để xây dựng các phương thức đó. Spring
