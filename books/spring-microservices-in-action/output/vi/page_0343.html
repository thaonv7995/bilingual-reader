10.6.2 Cài đặt công cụ trước khi build

Ồ, phần cấu hình trước khi build khá dài, nhưng mục tiếp theo thì ngắn. Các engine build thường là nguồn phát sinh một lượng lớn script “glue code” để kết nối các công cụ khác nhau được dùng trong quy trình build. Với script Travis của bạn, bạn cần cài đặt hai công cụ dòng lệnh:

  • Travis—Công cụ dòng lệnh này dùng để tương tác với build Travis. Bạn sẽ dùng nó ở phần sau trong chương để lấy token GitHub nhằm kích hoạt theo cách lập trình một build Travis khác.
  • ecs-cli—Đây là công cụ dòng lệnh dùng để tương tác với dịch vụ Amazon Elastic Container.

Mỗi mục được liệt kê trong phần before_install của tệp .travis.yml là một lệnh UNIX sẽ được thực thi trước khi build bắt đầu. Listing sau đây trình bày thuộc tính before_install cùng các lệnh cần chạy.

Sau khi cài đặt công cụ travis, bạn sẽ cài đặt công cụ Amazon ecs-cli. Đây là công cụ dòng lệnh dùng để triển khai, khởi động và dừng các container Docker chạy trên Amazon.
