Trong TARGET_URL, bạn truyền tham số truy vấn HTTP tên access_token. Tham số này chứa token truy cập cá nhân GitHub được thiết lập đặc biệt để cho phép script của bạn thực hiện hành động qua REST API. Token truy cập cá nhân GitHub của bạn được lưu trong biến môi trường được mã hóa tên GITHUB_TOKEN. Để tạo token truy cập cá nhân, đăng nhập vào tài khoản GitHub và truy cập https://github.com/settings/tokens. Khi tạo token, hãy copy và dán ngay lập tức. Khi rời màn hình GitHub, token sẽ không còn và bạn sẽ cần tạo lại.
