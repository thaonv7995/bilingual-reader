3.2.2 Sử dụng Spring Cloud configuration server với hệ thống tệp

Spring Cloud configuration server sử dụng một mục trong tệp confsvr/src/main/resources/application.yml để trỏ đến kho sẽ chứa dữ liệu cấu hình ứng dụng. Thiết lập kho dựa trên hệ thống tệp là cách dễ nhất để thực hiện việc này.

Để làm điều này, hãy thêm thông tin sau vào tệp application.yml của máy chủ cấu hình. Bảng liệt kê sau đây trình bày nội dung tệp application.yml của Spring Cloud configuration server.
