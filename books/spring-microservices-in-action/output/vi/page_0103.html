78 CHƯƠNG 3 Kiểm soát cấu hình với Spring Cloud configuration server

và một thuộc tính đơn giản sẽ được đọc từ Spring Cloud configuration server. Hình 3.6 cho thấy những gì sẽ xảy ra giữa licensing service và dịch vụ Spring Cloud configuration.

Khi licensing service được khởi động lần đầu, bạn sẽ truyền cho nó qua dòng lệnh hai thông tin: Spring profile và endpoint mà licensing service nên dùng để giao tiếp với dịch vụ Spring Cloud configuration. Giá trị Spring profile tương ứng với môi trường của các thuộc tính được truy xuất cho dịch vụ Spring. Khi licensing service khởi động lần đầu, nó sẽ liên hệ dịch vụ Spring Cloud Config qua một endpoint được xây dựng từ Spring profile được truyền vào. Dịch vụ Spring Cloud Config sau đó sẽ dùng kho cấu hình back end đã cấu hình (filesystem, Git, Consul, Eureka) để truy xuất thông tin cấu hình cụ thể cho giá trị Spring profile được truyền trên URI. Các giá trị thuộc tính phù hợp sau đó được trả về cho licensing service. Framework Spring Boot sau đó sẽ inject các giá trị này vào các phần thích hợp của ứng dụng.
