94 CHƯƠNG 3 Kiểm soát cấu hình với Spring Cloud configuration server

Vì licensing service giờ chịu trách nhiệm giải mã các thuộc tính đã mã hóa, trước tiên bạn cần thiết lập khóa đối xứng trên licensing service bằng cách đảm bảo biến môi trường ENCRYPT_KEY được đặt cùng khóa đối xứng (ví dụ, IMSYMMETRIC) mà bạn đã dùng với Spring Cloud Config server.

Tiếp theo, bạn cần thêm dependency JAR spring-security-rsa vào licensing service:

Các tệp JAR này chứa mã Spring cần thiết để giải mã các thuộc tính đã mã hóa được lấy từ Spring Cloud Config. Sau khi áp dụng các thay đổi này, bạn có thể khởi động Spring Cloud Config và các dịch vụ licensing. Nếu bạn gọi endpoint http://localhost:8888/licensingservice/default, bạn sẽ thấy spring.datasource.password được trả về ở dạng đã mã hóa. Hình 3.10 cho thấy kết quả từ lần gọi đó.
