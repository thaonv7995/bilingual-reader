Tổng hợp nhật ký và Spring Cloud Sleuth 273

Vì Zuul hiện đã được bật Spring Cloud Sleuth, bạn có thể truy cập thông tin tracing từ bên trong ResponseFilter bằng cách autowire lớp Tracer vào ResponseFilter. Lớp Tracer cho phép bạn truy cập thông tin về trace Spring Cloud Sleuth hiện đang được thực thi. Phương thức tracer.getCurrentSpan().traceIdString() cho phép bạn lấy dưới dạng String trace ID hiện tại của giao dịch đang diễn ra.

Việc thêm trace ID vào HTTP response đi ra đi qua Zuul rất đơn giản. Điều này được thực hiện bằng cách gọi

Khi đã có đoạn mã này, nếu bạn gọi một microservice EagleEye qua cổng Zuul, bạn sẽ nhận lại HTTP response có header tên tmx-correlation-id.
