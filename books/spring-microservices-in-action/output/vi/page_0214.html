class SimpleHostRoutingFilter. Mặc dù chúng ta sẽ không đi qua tất cả các hàm helper được gọi trong phương thức forwardToSpecialRoute(), chúng ta sẽ lần lượt xem xét code trong phương thức này, như trình bày trong listing sau đây.

Điểm then chốt từ code trong listing 6.18 là bạn đang sao chép tất cả các giá trị từ HTTP request đến (các tham số header, HTTP verb và body) vào một yêu cầu mới sẽ được gọi trên dịch vụ đích. Phương thức forwardToSpecialRoute() sau đó nhận phản hồi từ dịch vụ đích và gán vào HTTP request context mà Zuul sử dụng. Việc này được thực hiện qua phương thức helper setResponse() (không hiển thị). Zuul dùng HTTP request context để trả phản hồi về cho client dịch vụ gọi.
