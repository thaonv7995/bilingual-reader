Khi chạy lệnh này mà không đặt thuộc tính nào, licensing server sẽ tự động cố gắng kết nối tới máy chủ cấu hình Spring Cloud bằng endpoint (http://localhost:8888) và profile đang hoạt động (mặc định) được định nghĩa trong tệp bootstrap.yml của licensing service.

Nếu bạn muốn ghi đè các giá trị mặc định này và trỏ tới môi trường khác, bạn có thể làm điều này bằng cách biên dịch dự án licensingservice thành JAR rồi chạy JAR với ghi đè thuộc tính hệ thống -D. Lệnh dòng lệnh sau minh họa cách khởi động licensing service với profile không phải mặc định:

Với dòng lệnh trước, bạn đang ghi đè hai tham số: spring.cloud.config.uri và spring.profiles.active. Với thuộc tính hệ thống -Dspring.cloud.config.uri=http://localhost:8888, bạn đang trỏ tới máy chủ cấu hình chạy bên ngoài máy cục bộ của bạn.

LƯU Ý Nếu bạn thử chạy licensing service đã tải từ kho GitHub (https://github.com/carnellj/spmia-chapter3) từ máy tính bàn của bạn bằng lệnh Java trước đó, lệnh sẽ thất bại vì bạn không có máy chủ Postgres trên máy tính bàn đang chạy và mã nguồn trong kho GitHub đang dùng mã hóa trên máy chủ cấu hình. Chúng ta sẽ đề cập việc dùng mã hóa sau trong chương. Ví dụ trước minh họa cách ghi đè thuộc tính Spring qua dòng lệnh.

Với thuộc tính hệ thống -Dspring.profiles.active=dev, bạn đang báo cho licensing service dùng profile dev (đọc từ máy chủ cấu hình) để kết nối tới instance dev của cơ sở dữ liệu.

Dùng biến môi trường để truyền thông tin khởi động

Trong các ví dụ, bạn đang hard-code các giá trị truyền vào tham số –D. Trên cloud, hầu hết dữ liệu cấu hình ứng dụng bạn cần sẽ nằm trên máy chủ cấu hình. Tuy nhiên, đối với thông tin bạn cần để khởi động dịch vụ (ví dụ dữ liệu cho máy chủ cấu hình), bạn sẽ khởi động instance VM hoặc container Docker và truyền vào biến môi trường.

Tất cả ví dụ mã cho mỗi chương đều có thể chạy hoàn toàn trong container Docker. Với Docker, bạn mô phỏng các môi trường khác nhau thông qua các tệp Docker-compose theo môi trường điều phối việc khởi động mọi dịch vụ của bạn. Các giá trị theo môi trường cụ thể mà container cần được truyền vào dưới dạng biến môi trường
