219 JavaScript Web Tokens và OAuth2

xem ví dụ về class JWTTokenStoreConfig ở cả hai class licensing-service/src/main/com/thoughtmechanix/licensing-service/security/JWTTokenStoreConfig.java và organization-service/src/main/com/thoughtmechanix/organization-service/security/JWTTokenStoreConfig.java.

Bạn cần làm thêm một phần việc cuối cùng. Vì licensing service gọi organization service, bạn cần đảm bảo rằng OAuth2 token được truyền đi. Việc này thường được thực hiện qua class OAuth2RestTemplate; tuy nhiên, class OAuth2RestTemplate không truyền token dựa trên JWT. Để đảm bảo licensing service của bạn thực hiện việc này, bạn cần thêm một bean RestTemplate tùy chỉnh sẽ thực hiện việc inject này cho bạn. RestTemplate tùy chỉnh này có thể tìm thấy trong class licensing-service/src/main/java/com/thoughtmechanix/licenses/Application.java.

Listing sau đây cho thấy định nghĩa bean tùy chỉnh này.

Trong code trước đó, bạn đang định nghĩa một bean RestTemplate tùy chỉnh sẽ sử dụng ClientHttpRequestInterceptor. Hãy nhớ lại từ chương 6 rằng ClientHttpRequestInterceptor là class Spring cho phép bạn gắn thêm chức năng để thực thi trước khi một lệnh gọi dựa trên REST được thực hiện. Class interceptor này là biến thể của class UserContextInterceptor mà bạn đã định nghĩa ở chương 6. Class này nằm trong licensing-service/src/main/java/com/thoughtmechanix/licenses/utils/UserContextInterceptor.java. Listing sau đây cho thấy class này.
