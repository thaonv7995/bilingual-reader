Việc đầu tiên bạn làm trong listing 10.8 là dùng công cụ dòng lệnh Travis CI để đăng nhập Travis CI và lấy token OAuth2 dùng cho các Travis REST API khác. Bạn lưu token OAuth2 này vào biến môi trường $RESULTS.

Tiếp theo, bạn xây dựng body JSON cho lời gọi REST API. Job Travis CI phía sau sẽ khởi chạy một loạt script Python kiểm thử API của bạn. Job phía sau này yêu cầu hai biến môi trường được thiết lập.
