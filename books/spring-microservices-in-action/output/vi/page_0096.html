Phiên bản Spring Cloud sẽ được dùng

Lớp bootstrap sẽ dùng cho configuration server

Các dự án Spring Cloud bạn sẽ dùng trong dịch vụ cụ thể này

Trong tệp Maven trong listing trước, bạn bắt đầu bằng cách khai báo phiên bản Spring Boot sẽ dùng cho microservice của mình (phiên bản 1.4.4). Phần quan trọng tiếp theo trong định nghĩa Maven là BOM (Bill of Materials) cha của Spring Cloud Configuration mà bạn sẽ dùng. Spring Cloud là tập hợp lớn gồm nhiều dự án độc lập, mỗi dự án phát hành theo lịch riêng. BOM cha này chứa mọi thư viện bên thứ ba và dependency được dùng trong dự án cloud, cùng số phiên bản của từng dự án con tạo nên phiên bản đó. Trong ví dụ này, bạn dùng Spring Cloud phiên bản Camden.SR5. Nhờ định nghĩa BOM, bạn có thể đảm bảo dùng các phiên bản tương thích của các dự án con trong Spring Cloud.
