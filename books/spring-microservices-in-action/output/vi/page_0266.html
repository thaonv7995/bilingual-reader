Trong listing này, bạn đang dùng interface Source. Interface Source là một interface được Spring Cloud định nghĩa, expose một phương thức duy nhất gọi là output(). Interface Source là một interface tiện lợi khi dịch vụ của bạn chỉ cần publish lên một channel duy nhất. Phương thức output() trả về một class kiểu MessageChannel. MessageChannel là cách bạn gửi message tới message broker. Sau trong chương này, tôi sẽ cho bạn thấy cách expose nhiều messaging channel bằng một custom interface.
