Xây dựng bộ lọc pre-filter Zuul đầu tiên tạo ID tương quan 179

Bộ lọc của bạn lấy correlation ID từ header và đặt giá trị đó lên class UserContext.

Các giá trị khác được lấy từ HTTP Headers sẽ phát huy tác dụng nếu bạn dùng ví dụ dịch vụ xác thực được định nghĩa trong file README của mã nguồn.

Cuối cùng, UserContextFilter được dùng để ánh xạ các giá trị HTTP header mà bạn quan tâm vào một class Java, UserContext.

UserContext: Giúp các HTTP header dễ truy cập cho dịch vụ

Class UserContext được dùng để giữ các giá trị HTTP header cho yêu cầu client dịch vụ riêng lẻ đang được microservice của bạn xử lý. Nó gồm phương thức getter/setter để lấy và lưu giá trị từ java.lang.ThreadLocal. Listing sau đây cho thấy mã từ class UserContext. Mã nguồn của class này có tại licensing-service/src/main/java/com/thoughtmechanix/licenses/utils/UserContext.java.
