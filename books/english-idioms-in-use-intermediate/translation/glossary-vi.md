# Glossary - English Idioms In Use Intermediate (EN-VI)

## Naming Conventions
- Keep the English idiom phrases exactly as they are in the source, but explain their Vietnamese equivalents in bracketed text or in the prose.
- Example: **a blessing in disguise** -> **a blessing in disguise** (trong cái rủi có cái may)

## Technical / Study Terms
- Exercise -> Bài tập
- Idiom -> Thành ngữ
- Meaning -> Ý nghĩa / Nghĩa
- Context -> Ngữ cảnh
- Example -> Ví dụ
- Section -> Phần
- Unit -> Bài học / Unit
- Reference -> Tham chiếu

## Style Guidelines
- Maintain natural, fluent, and pedagogical Vietnamese.
- Preserve all formatting (bolding, italics, list layouts).
